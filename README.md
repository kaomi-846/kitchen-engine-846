# 🏭 Kitchen Engine 846  
### 農産工房やしろ 製造・原価・レシピ統合システム

---

## 🌿 Overview

**Kitchen Engine 846** は、小規模加工場・農産工房向けの  
製造記録・レシピ管理・原価計算・利益分析を統合したシステムです。

現場の「なんとなくの原価」をなくし、  
製造単位ごとの利益構造を見える化します。

---

## ⚙️ Core Concept

> 小さな工房に、確かな原価エンジンを。

- レシピ（BOM）から自動原価計算
- g / ml / 個 / kg / L の単位自動変換
- ロット単位での利益管理
- 現場入力と経営判断の分離

---

## 🍋 Features

### 🧾 原材料管理
- 規格単位付きマスタ管理
- 購入価格・単価自動計算
- 購入先の記録

### 📖 レシピ（BOM）
- 原材料を組み合わせたレシピ管理
- 中間原料（ピクルス液など）対応
- 再利用可能な構造

### 🏷 製品管理
- 製品ごとの売価設定
- レシピとの紐付け

### 📦 ロット管理
- 日付ベースのロット自動生成
- 製造ごとの原価・利益計算
- 松竹梅＋桐レベル評価

### ⚖ 単位統一エンジン
- kg → g
- L → ml
- 個体管理対応

---

## 📊 Purpose

このシステムは以下を目的としています：

- 原価の見える化
- 小規模工房の経営判断支援
- レシピの再現性向上
- 利益構造の可視化

## 🚀 How to Use

1. ブラウザで開く（GitHub Pages対応）
2. 原材料を登録
3. レシピを作成
4. 製品を登録
5. ロットを作成して製造記録
